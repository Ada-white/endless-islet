# Single Dragon Shape Design

## Goal

Dragon caves should show one giant dragon with a readable body, not a cluster of small dragon-like symbols.

## Requirements

- Each dragon cave renders only one dragon character.
- Eastern dragons are one coiled body: a continuous looped spine, one raised head, two horns, eyes, whiskers, mane, visible scales, tail, and claws.
- Western dragons remain one standing winged body: one head, one torso, one tail, one pair of wings, one set of claws, horns, eyes, chest plates, and scales.
- The dragon should keep the existing pixel/canvas style and subtle breathing motion.
- Existing species, names, speech, gift chances, teleport targets, and interactions stay unchanged.

## Creative Names

- Eastern silhouette: **Single Coiling Long**
- Eastern atmosphere: **Cloud-Wreathed Long**
- Western silhouette: **Lone Cavern Drake**
