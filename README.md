# Endless Islet

《无尽小岛》是一个像素探索小游戏。玩家可以在岛屿、海滩、森林、洞窟和龙洞之间漫游，收集物品，记录记忆，并遇见不同气质的东方龙和西方龙。

## Play

This project is a static web game. Open `index.html` in a browser, or publish the repository with GitHub Pages.

## Files

- `index.html` - game shell and UI
- `style.css` - visual layout and HUD styling
- `game.js` - world generation, rendering, interactions, memory, inventory, teleport, caves, and dragons
- `tests/` - lightweight content checks for important game rules
